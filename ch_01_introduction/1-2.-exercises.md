**Ex 3.** 다음의 데이터 세트에 대한 도수 분포표를 작성하라.

```text
22 25 22 27 24 23 26 24 22 24 26
```

 **\[Solution\]**

{% tabs %}
{% tab title="R Source" %}
```text
x <- c(22, 25, 22, 27, 24, 23, 26, 24, 22, 24, 26)
table(x)
```
{% endtab %}

{% tab title="Result" %}
```text
 table(x)
## x
## 22 23 24 25 26 27 
##  3  1  3  1  2  1
```
{% endtab %}
{% endtabs %}

**Ex 4.** 다음의 데이터 세트에 대한 도수 분포표를 작성하라.
