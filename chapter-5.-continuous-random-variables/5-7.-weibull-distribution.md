# 5-7. Weibull Distribution

와이블 분포는 수명분포로 가장 많이 사용된다.



> **Weibull Distribution**
>
> $$f(x) = \left ( \frac{\alpha}{\theta}  \right  ) \left ( \frac{x}{\theta}  \right  ) ^{\alpha -1}exp\left [ - \left ( \frac{x}{\theta} \right )^\alpha \right ], $$     $$x > 0;$$   $$\alpha>0, $$  $$\theta>0.$$
